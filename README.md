# 🚆 Yathrava

AI-Powered Railway Travel Assistant designed to make train journeys smarter, safer, and more convenient.

## ✨ Features

- 🎙️ AI Voice Assistant (English, Tamil & Tanglish)
- 🚉 Train Location & Journey Guidance
- 🔔 Smart Station Alerts
- 🎫 Digital Ticket & PNR Management
- 📷 OCR-Based Ticket & Station Scanner
- 🚨 Emergency Safety Assistance
- 👤 User Profile & Preferences
- 📱 Mobile & Web Responsive Design

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Firebase Authentication

### Backend
- Flask
- MongoDB
- JWT Authentication
- REST APIs

### Deployment
- Vercel (Frontend)
- Render (Backend)

## 🎯 Purpose

Yathrava helps railway passengers by providing intelligent travel assistance, timely alerts, multilingual voice support, and enhanced journey safety through AI-powered features.

## 🚀 Key Benefits

- Prevent missed stations
- Easy access to travel information
- Multilingual assistance
- Improved passenger safety
- Smart railway experience
