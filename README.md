Yathrava – AI-Powered Railway Travel Assistant 🚆🤖
Overview

Yathrava is an intelligent railway travel companion designed to make train journeys safer, smarter, and more comfortable. The platform combines AI assistance, real-time travel guidance, station alerts, smart ticket management, emergency support, and multilingual voice interaction into a single user-friendly application.

Built with a modern React + Vite frontend and a Flask backend, Yathrava helps passengers stay informed throughout their journey while providing accessibility-focused features for all types of travelers.
